# Unicity Claude Marketplace

Official Claude Code plugin marketplace for the Unicity Sphere wallet ecosystem.

## Installation

In Claude Code (VS Code extension):

1. Open **Manage Plugins** → **Marketplaces** tab
2. Add: `https://github.com/unicity-sphere/unicity-claude-marketplace`
3. Switch to **Plugins** tab and enable the plugins you need

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [sphere-connect](https://github.com/unicity-sphere/sphere-connect-plugin) | Integrate Sphere wallet Connect protocol into your dApp |

## For Plugin Developers

To add a plugin to this marketplace, submit a PR adding your plugin entry to `.claude-plugin/marketplace.json`.
